# Odin Recipes

Proyecto de práctica del curso Foundations de The Odin Project.

Una colección de recetas colombianas caseras construida con HTML puro.

## Recetas incluidas
- Ajiaco Santafereño
- Bandeja paisa
- Changua
- Arepa con huevos pericos ¡y mucho queso!

## Tecnologías usadas
- HTML5

## Sobre este proyecto

Segundo proyecto del curso Foundations de The Odin Project.
Construido en 4 días como parte de mi aprendizaje de desarrollo web.
