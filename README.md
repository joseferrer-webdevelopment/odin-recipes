# Odin Recipes

Proyecto de práctica del curso Foundations de The Odin Project.

Una colección de recetas colombianas caseras construida con HTML puro.

## Recetas incluidas
- Ajiaco Santafereño
- Bandeja paisa
- Changua
- Moñona
- Arepa con huevos pericos ¡y mucho queso!

## Tecnologías usadas
- HTML5
